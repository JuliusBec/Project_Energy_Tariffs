Usage
=====

Running the Application
-----------------------

To start the Flask application:

.. code-block:: bash

    python app.py

Or using the start script:

.. code-block:: bash

    ./start.sh

The application will be available at ``http://localhost:5000``.

Basic Usage
-----------

The Energy Tariffs platform provides several key features:

Tariff Analysis
~~~~~~~~~~~~~~~

Analyze and compare different energy tariff structures to determine
the most cost-effective option for your usage pattern.

Risk Analysis
~~~~~~~~~~~~~

Evaluate the financial risks associated with different tariff models,
including volatility analysis and worst-case scenarios.

Usage Forecasting
~~~~~~~~~~~~~~~~~

Generate forecasts for future energy consumption based on historical
usage patterns and external factors like temperature.

Price Forecasting
~~~~~~~~~~~~~~~~~

Predict future energy market prices using various forecasting models.
